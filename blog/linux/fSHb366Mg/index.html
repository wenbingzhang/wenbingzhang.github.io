    ssh客户端于服务端保持连接

    2021-06-11   |   Linux   |   Linux

    linux ssh 默认没有开启心跳，所以很容易导致连接断开。

    服务端设置 #

    sudo vim /etc/ssh/sshd_config

TCPKeepAlive yes
ClientAliveInterval 60
ClientAliveCountMax 3

    客户端设置 #

    一般还是建议在客户端设置。

    sudo vim /etc/ssh/ssh_config

TCPKeepAlive yes
ServerAliveInterval 60
ServerAliveCountMax 3
