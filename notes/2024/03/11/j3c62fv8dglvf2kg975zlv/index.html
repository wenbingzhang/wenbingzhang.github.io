    📝 标签操作
    • 创建标签
    • 推送标签到远程仓库
    • 推送所有标签到远程仓库
    • 查看本地所有标签
    • 查看远程所有标签
    • 删除本地或远程标签

    创建标签 #

    git tag <tagname>

    推送标签到远程仓库 #

    git push origin <tagname>

    推送所有标签到远程仓库 #

    git push origin --tags

    查看本地所有标签 #

    git tag

    查看远程所有标签 #

    git ls-remote --tags origin

    删除本地或远程标签 #

    git tag -d <tagname>

# 删除远程标签
git push origin :refs/tags/<tagname>
